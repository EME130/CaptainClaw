# Contributor Onboarding

## /godmode:choose-your_way
- Command at `.claude/commands/godmode/choose-your_way.md`
- Detects `git config user.name`, lets contributor pick their commit heading style
- Stores preference in `LootChest/<NAME>/CommitStyle.md`

## Commit Style System
- Each contributor owns their heading convention
- Claude reads `CommitStyle.md` before formatting any commit
- eme uses leet-speak: `Add L00tCh3st`, `R3m0v3 pr0pr13tary T1TL3.wav fr0m tr4ck1ng`

## Conventions
- Unix philosophy: no whitespace in filenames
- Commit messages are for dummies — keep them fun/stylized
