# Commit Style

## Convention
Leet-speak / creative encoding: replace letters with numbers, use unicode or special characters for file extensions and words. The heading should be decipherable but stylized.

- `Add L00tCh3st`
- `Add R0AdMa9.碌鈭俙
- `Remove proprietary TITLE.wav from tracking` would become something like `R3m0v3 pr0pr13tary T1TL3.wav fr0m tr4ck1ng`
